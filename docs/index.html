施術メニュー紹介

国家資格者による丁寧な施術。まずは気軽にご相談ください。

  • マッサージ

    筋肉の緊張を和らげ、日常の疲労回復や慢性的なコリの緩和をサポートします。問診→触診→筋肉のほぐしで対応します。

  • 骨盤矯正

    骨盤のバランスを整え、姿勢や体型の崩れ、産後の不安定さにアプローチします。問診→状態チェック→手技で調整します。

  • ストレッチ・猫背矯正

    胸まわりと背中の柔軟性を高め、猫背を改善します。姿勢チェック→筋膜リリースやストレッチで整えます。

  • 肩甲骨はがし

    肩甲骨周りの硬さをほぐし、肩の可動域を広げて動作を楽にします。丁寧なほぐしと可動域確認で対応します。

  • ドライヘッド矯正

    ヘッドスパで頭の筋肉を緩めるとともに、23個の骨のゆがみを調整します。顔や頭の状態確認→首の筋肉を丁寧にほぐす→パーツごとにまっすぐに整えます。
